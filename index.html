My work

CarPe Diem

CarPe-Diem is an object-oriented PHP project that follows SOLID principles and is styled with SASS. It fetches and displays information about different cars from a MySQL database. It also employs unit testing. The database is built and populated using a utility tool that curls the data from an existing API. My focus was the utility tool as well as the filtering of information according to the make and/or colour of the car.

Languages / Libraries

  • PHP
  • SASS
  • SQL
  • HTML

Competencies

  • - OOP
  • - Unit testing
  • - SOLID principles

Scran Master

Scran Master is a React and Bootstrap based web-app for ordering takeaways online. It fetches restaurant data from an API. You can add and remove items from your order. It involves a simulation of placing an order during which the server will respond with an estimated delivery time. I was the Scrum Master for this project.

Languages / Libraries

  • React
  • JavaScript
  • HTML
  • Bootstrap

Competencies

  • - APIs
  • - Libraries
  • - Postman
