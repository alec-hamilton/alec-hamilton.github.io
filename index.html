Scran Master

Scran Master is a React and Bootstrap based web-app for ordering takeaways online. It fetches Restaurant data from an API. You can add and remove items from your order as well as simulate placing it, where the server will respond with an estimated delivery time. I worked on populating the place order section when someone adds or removes an item.

Languages / Libraries

  • JavaScript
  • HTML
  • Bootstrap

Competencies

CarPe Diem
